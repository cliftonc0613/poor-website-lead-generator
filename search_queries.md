## üîç Advanced Search Operators

### Basic Operators
- `site:domain.com` - Search within specific site
- `inurl:word` - Word appears in URL  
- `filetype:htm` - Specific file types
- `intitle:"phrase"` - Exact phrase in title
- `-word` - Exclude word/site
- `"exact phrase"` - Must contain exact phrase

### Poor Website Indicators
```
inurl:index.html          # Default index pages
inurl:default.asp         # Default ASP pages  
inurl:placeholder         # Placeholder content
inurl:demo               # Demo/test sites
inurl:template           # Template sites
inurl:~                  # Personal directories
filetype:swf             # Flash content
```

## üí° Pro Tips

### Search Strategy
1. **Start broad** - Use basic service + location
2. **Add exclusions** - Remove directory sites  
3. **Look for tech issues** - Add file type or URL indicators
4. **Check manually** - Verify 5-10 results before automating

### Time-Saving Tricks
- **Use quotes** for exact phrases
- **Exclude directories** with `-site:domain.com`
- **Search file types** with `filetype:extension`
- **Check URLs** with `inurl:keyword`

### Red Flags to Watch For
- HTTP instead of HTTPS
- Old copyright dates
- "Under construction" anywhere
- Flash player requirements  
- Mobile display issues
